# mountain-dew

마운틴 듀 웹사이트 스프링 버전입니다. 

## 정보

* 팀 이름 => 마운틴듀(산이슬)

팀원
- 정영운 : backend(spring), frontend(javascript, mustache)
- 유수빈 : db 구축(mysql), backend(spring)
- 한종훈 : db 구축(postgresql), frontend(javascript, mustache)

## Stack
- IDE : vscode, pycharm
- frontend : html, css, javascript
- backend : django, node.js
- DB : mysql, postgresql
- tool : wix, miro
